# Week2

## 1. 분류(Classification) vs 회귀(Regression)

지도학습은 예측하려는 타겟 데이터의 형태에 따라 다음 두가지로 나뉩니다.

- **분류**: 정답데이터가 0또는 1과 같이 이진적인 값일 떄 사용합니다. 모델은 입력 데이터가 정답인지 아닌지 예측합니다.
- **회귀**: 정답데이터가 연속적인 수치일때 사용합니다. 모델은 입력 데이터가 가지는 정답 수치를 예측합니다.

## 2. 분류

#### 2.1 모델

1. Logistic Regression
2. Decision Tree Classifier
3. Random Forest Classifier

#### 2.2 성능 평가 지표

**1. 정확도**  
- 전체 예측 중 맞춘 비율을 의미합니다.  
  
**2. Confusion Matrix**  
- 실제 정답과 모델이 예측한 값을 표 형식으로 보여줍니다.
- [[1인 경우 1을 예측, 1인데 0을 예측],[ 0인데 1을 예측, 0인데 0을 예측]]의 형식으로 보여줍니다.
     
     

## 3. 회귀 

#### 3.1모델

1. Linear Regression
2. Decision Tree Regressor
3. Random Forest Regressor

#### 3.2 성능 평가 지표

**1. MSE (평균제곱오차)**  
- 실제값과 예측값의 차이(오차)를 제곱하여 평균을 낸 값,  
  값이 0에 가까울수록 모델의 예측이 실제값에 가깝다는 것을 의미합니다.
